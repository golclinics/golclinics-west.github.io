Software Engineer and Program Manager
— from zero to hero.

An intensive 3-month program to help university students from all walks of life and beginner developers to level up on
.Net, JavaScript, Git, Azure, Software Design and Program Manager Fundamentals.

Pre-requisites

  • Learners should have access to a computer/laptop for the duration of the program.
  • At least 80% attendance is required for graduation.

Core Topics

Developer Path: Azure Basics, .Net, JavaScript, Basic Solution Design, Git Best Practices

Program Manager Path: Product Requirement Documentation, Project Management 101, Prioritization Framework

Learners will be guided along both paths, but will have the opportunity to specialize during their team-based project work.

Schedule

The program begins in October 2021. Recordings for all sessions are available to learners in Microsoft Teams.

Sessions run every Tuesday and Thursday, 5.00 - 6.30 pm, WAT.
